## ✨ Features

- **Add / Edit / Delete** students with up to 3 subjects each
- **Live Dashboard Stats** — Total students, Class average, Highest mark, Pass rate
- **Grade Badges** — Automatic A / B / C / D / F grading based on average
- **Color-coded Marks** — Green (good), Amber (average), Red (below pass)
- **Progress Bars** — Visual representation of each student's performance
- **Search & Filter** — Filter by name, ID, subject, or grade
- **Sort** — By name, average (ascending/descending), or student ID
- **Score Distribution Chart** — Bar chart across 5 score bands
- **Grade Distribution** — Breakdown of how many students fall in each grade
- **CSV Export** — Download the full gradebook as a `.csv` file
- **Persistent Storage** — Data saved via `localStorage`, survives page refresh

## 📋 How to Use

1. Click **+ Add Student** to enter a student's name, ID, subject, and marks
2. Optionally add up to 3 subjects per student
3. The dashboard stats and charts update **automatically**
4. Use the **search bar** to find a student by name or ID
5. Use the **filter dropdowns** to narrow by subject or grade
6. Click **✎** to edit a student or **✕** to delete
7. Click **↓ Export** to download all data as a CSV file
